# Architecture

## Backend
- ASP.NET Core Web Api
- Target framework: .NET 10
- Program.cs + Startup.cs pattern
- MVC controllers (not minimal APIs)
- Stable ASP.NET Core patterns only

## Frontend
- Angular
- Standalone components
- Typed HttpClient services

## Rules
- No invented frameworks
- No speculative or preview-only APIs
- No architectural refactors without explicit instruction
